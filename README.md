# Part 1: Business Data Cleaning, Validation & Excel Reporting

## Project Overview
This repository contains the completion of Part 1 of the Business Analytics Capstone Project. The objective of this phase is to take raw, messy order-level sales data from multiple internal systems, clean it, validate it against business rules, and produce analysis-ready reports.

## Folder Structure
* **`data/`**: Contains the original raw dataset (`raw_orders.xlsx`) and the final cleaned dataset (`cleaned_orders.xlsx`).
* **`outputs/`**: Contains the data quality report, pivot table summaries, and the detailed markdown log of all cleaning steps performed.
* **`screenshots/`**: Contains visual proofs of the raw data state, the cleaned data state, and the pivot table summaries.

## Tools Used
* Microsoft Excel (Data Cleaning, Validation, Power Query/Formulas, Pivot Tables)
* GitHub (Version Control and Submission)
